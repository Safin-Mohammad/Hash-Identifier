## Installation

For debian based linux os

```bash
sudo apt install python3 -y
git clone https://github.com/Safin-Mohammad/Hash-Identifier
cd Hash-Identifier
python3 Hash-Identifier.py
```
For termux

```bash
pkg install python -y
pkg install git -y
git clone https://github.com/Safin-Mohammad/Hash-Identifier  
cd Hash-Identifier  
python Hash-Identifier.py
```

For arch based linux

```bash
sudo pacman -Sy python3 -y  
git clone https://github.com/Safin-Mohammad/Hash-Identifier  
cd Hash-Identifier  
python3 Hash-Identifier.py
```
